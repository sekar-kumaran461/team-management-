How to Use Recurring Tasks

For Team Members:

  • • Visit the Recurring Tasks page
  • • Choose which daily/weekly tasks you want
  • • Tasks will be automatically created for you
  • • Complete them like regular tasks

Automatic Generation:

  • • Tasks are generated automatically each day
  • • Use the "Generate Today's Tasks" button for manual generation
  • • Set up a cron job for: python manage.py generate_recurring_tasks
  • • Each instance is separate and trackable
